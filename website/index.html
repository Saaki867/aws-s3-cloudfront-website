This is a learning project for AWS Cloud.

About This Project

This website is hosted on Amazon S3 and delivered via CloudFront CDN.

It demonstrates how to create a secure, high-performance static website using AWS services.

  • ✔ Amazon S3 for storage
  • ✔ CloudFront for content delivery
  • ✔ HTTPS for security
  • ✔ CloudWatch for monitoring

AWS Services Used

S3

Simple Storage Service for hosting website files

CloudFront

Content Delivery Network for global distribution

Certificate Manager

SSL/TLS certificate management

CloudWatch

Metrics and monitoring
